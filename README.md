# probable-meme

This Master Branch shows node_modules. This is bad. Check out the other branch to see node modules gone



To remove node_modules
- `touch .gitignore` at root of project
In **.gitignore**
  - node_modules
  - or copy paste the class repo's `.gitignore` file
- `rm -rf node_modules`
- `git add .`
- `git commit -m 'removed node modules'`
- `npm i`
